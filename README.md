# Employee Performance Analytics Project

**IABAC Final Certification Project**

---

## üìå Project Overview

This project was developed as part of the **final certification examination conducted by IABAC (International Association of Business Analytics Certification)**. The objective was to analyze employee performance data of **INX Future Inc.** and extract meaningful, data-driven business insights.

The project follows an **end-to-end data analytics workflow**, focusing on understanding performance drivers, identifying improvement areas, and supporting managerial decision-making.

---

## üéØ Business Problem

INX Future Inc. aimed to improve overall employee performance but lacked clarity on:

* Key factors influencing employee performance
* Department-wise and role-wise performance gaps
* Actionable insights for performance improvement

The task was to analyze historical employee data and provide insights that management could use for strategic decisions.

---

## üß† Methodology

The project was executed using a structured analytics approach:

### 1. Data Understanding

* Employee demographics
* Job roles and departments
* Performance ratings and workplace attributes

### 2. Data Cleaning & Preprocessing

* Handling missing values
* Removing irrelevant variables
* Ensuring data consistency

### 3. Exploratory Data Analysis (EDA)

* Department-wise performance comparison
* Role-based performance trends
* Impact of salary, work-life balance, and job satisfaction

### 4. Visualization & Insights

* Clear visual representation of findings
* Identification of major performance drivers
* Business-oriented interpretation of results

---

## üìä Key Insights

* Employee performance is strongly influenced by **job role, department, and work environment**
* Salary alone is **not a decisive factor** for high performance
* **Work-life balance and job satisfaction** show strong correlation with performance
* Certain departments require **targeted training and engagement strategies**
